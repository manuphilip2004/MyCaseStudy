Project Name: MyRetail

Project Detail / Agile user Story:
Responds to an HTTP GET request at /products/{id} and delivers product data as JSON (where {id} will be a number. 
Example product IDs: 15117729, 16483589, 16696652, 16752456, 15643793) 
Example response: {"id":13860428,"name":"The Big Lebowski (Blu-ray) (Widescreen)","current_price":{"value": 13.49,"currency_code":"USD"}}

Project Tech Stack:
Java 1.8 Spring boot 4.3.2.RELEASE
MongoDB version: '1.9.3.RELEASE
Gradle Build File: 1.4.0.RELEASE
Junit test framework 4.12
IntelliJ Idea community edition

Pre-Requisite:
Install the mongoDB.
localhost:8080
Mongodb listening port 27017
Database will be populated in the @PostConstruct annotation in the controller.
